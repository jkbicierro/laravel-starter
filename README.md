# laravel-starter 
a starter pack template for laravel 12.x

## prerequisites
- [php (ver8.0^)](https://www.php.net/)
- [composer (dependency manager for php)](https://getcomposer.org/)
- node.js

## getting started
### install dependencies
```bash
npm install
```

### build production
```bash
npm run build
```

### development server
```bash
composer run dev
```
